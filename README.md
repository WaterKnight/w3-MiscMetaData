# w3-MiscMetaData
Enhanced version of MiscMetaData.slk

# Usage
1. Activate Local Files (https://www.hiveworkshop.com/threads/local-files.330849/)
2. Put the MiscMetaData.slk file in Warcraft III/\_retail\_/units
3. Restart World Editor
